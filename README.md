# KostKita Finance - Manajemen Keuangan Anak Kos

Mini project Spring Boot microservices untuk catat pengeluaran, lihat sisa hidup, dan reminder kost.

## Tech Stack
- Spring Boot 3.1 + JPA
- Java Stream (grouping, reducing)
- Native SQL Query (JSON agg untuk dashboard)
- JWT Security
- Docker + PostgreSQL

## Jalankan
```bash
docker-compose up --build
